# The Exchange Preview Site

Files:
- `index.html` — page structure and approved web copy
- `style.css` — responsive SWCT Tech Hub visual styling
- `script.js` — mobile navigation behavior
- `assets/` — provided brand reference image

## Preview locally
Open `index.html` in a browser.

## Publish to GitHub Pages
Upload the contents of this folder to the intended repository and enable GitHub Pages for the branch/folder you want to publish.

## Forms
